Kanban Metrics Tool for miro

Basic Instructions

  • Create Shapes for the steps in your kanban workflow.
  • Place names for this steps in each chape in the format [<name>].
  • Move card widgets over the prepared shapes.
  • The metrics are added to the description of the cards automatically.
