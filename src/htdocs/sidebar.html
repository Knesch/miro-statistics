Kanban Statistics Plugin

To collect statistics in the background:

  • Create shapes for Kanban Lanes.
  • Mark these shapes with [<name>].
  • drag card widgets on this shapes.

The timestamps and the name of the shape will be added into a metrics block in the description of the card.

Date Constraints

Initially, all events before the start date or after the end date are removed. These fields are optional.

Show LTD Chart

Takes all Cards where the last event is the placement in the definied done column into a lead time distribution chart. The lead time of each card is the timespan from the first event until the placement in the done column. You can ignore columns by listing them in the corresponding textarea.
